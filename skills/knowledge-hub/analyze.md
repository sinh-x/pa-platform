# Knowledge Hub Analyze

Perform blind-spot and cadence analysis from current learning activity.

## Workflow

1. Read current tracker and latest gather/curate outputs.
2. Determine cadence context (daily/weekly/monthly/etc.).
3. Identify recurring gaps, neglected domains, and over-indexed areas.
4. Propose targeted corrective actions.

## Output

Write an analysis report with:

- cadence decision and evidence
- detected blind spots
- ranked improvement actions
