# Knowledge Hub Facilitate

Run interactive learning support based on user intent.

## Interaction Modes

- Quiz mode
- Briefing mode
- Q&A mode
- Live learning companion mode

## Workflow

1. Infer the requested mode from prompt/context.
2. Pull only relevant knowledge-base material.
3. Keep output structured, practical, and conversational.
4. End with suggested next learning actions.

## Output

Produce an interaction summary note in deployment workspace (topic, mode, key responses, follow-ups).
