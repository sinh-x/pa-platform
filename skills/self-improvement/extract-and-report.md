# Self-Improvement Extract And Report

Extract improvement suggestions from session logs and produce actionable reports.

## Workflow

1. Read target session logs for the selected time window.
2. Extract items from self-improvement sections.
3. Deduplicate and classify by scope (`skill`, `team`, `infra`, `prompt`).
4. Rank by impact and effort.
5. Emit report with concrete recommendations.

## Output

A structured report containing:

- new items
- recurring patterns
- top actions for next cycle
