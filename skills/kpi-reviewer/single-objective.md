# KPI Reviewer Single Objective

Evaluate one deployment against the KPI framework in `skills/requirements/kpi-definitions.md`.

## Required Flow

1. Locate deployment artifacts and session log.
2. Score against defined KPI dimensions and evidence anchors.
3. Provide concise rationale per score and an overall rating.
4. Identify any follow-up actions or verification gaps.

Deliver one structured evaluation report for that deployment.
