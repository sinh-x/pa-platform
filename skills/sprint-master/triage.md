# Sprint-Master Triage

Run ticket triage for backlog and in-flight work.

## Workflow

1. Review unassigned and pending tickets.
2. Validate status, priority, estimate, and assignee.
3. Correct obvious metadata drift and missing handoff info.
4. Flag blocked items and capture unblock actions.

## Output

Post concise triage updates on touched tickets and produce a short triage summary.
